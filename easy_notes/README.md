## Description
Easy Notesはメモの管理ができるサービスです。書いたメモはzip形式かtar形式でまとめて出力できます。
実験的なサービスなのでバグがたくさんあるかもしれませんが…もし見つけても悪用しないでくださいね。

Easy Notes is a note-taking service. You can write notes and export them as a .zip or .tar!
